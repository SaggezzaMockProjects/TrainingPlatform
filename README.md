# Saggezza Institute

## Information
 * This application is intended as use as a training platform.
 * More information for the application is on the company's Google Drive


## Build & development
 * We used firebase as for our realtime cloud database.
 * https://www.firebase.com/
 * AngularFire is the API used to communicate with firebase.
 * https://www.firebase.com/docs/web/libraries/angular/
 * Also used bootstrap for UI.
 * We also used a local server called 'lite-server'
 * https://github.com/johnpapa/lite-server

## Testing
 * No automated tests were performed.
